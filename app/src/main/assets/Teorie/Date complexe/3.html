Funcții pentru sir de caractere

Funcțiile pentru operatii cu șiruri se găsesc in header-ul <string.h>.

Funcția strlen returnează lungimea șirului. Caracterul nul nu se numără. 

cout << strlen("informatica"); // 11

Funcția strcpy copiază caracterele dint-un șir in alt șir inclusiv caracterul nul. 

char sursa[5] = "info";
char destinatie[5];
strcpy(destinatie , sursa); // se muta sirul din variabila sursa in variabila destinatie
cout<< destinatie; // se afiseaza info

Funcția strncpy copiază primele n caracterele dint-un șir in alt șir.

char s[100]="informatica";
strncpy(s, "test", 3);
cout << s; // tesormatica

Funcția strcat adaugă caracterele dint-un șir inclusiv caracterul nul, la alt șir. 

char a[21]="info", b[21] = "bac";
strcat(a , b);
cout << a; // infobac

Strchr caută caracterul în șir. Dacă îl găsește returnează șirul începând cu prima apariție a caracterului altfel returnează null. 

char ch = 'i';
if(strchr("aeiou" , ch) != NULL)
    cout << "VOCALA"
else
    cout << "CONSOANA";
//se va afisa VOCALA

Strstr caută un șir în alt șir. Dacă îl găsește returnează șirul începând cu prima apariție a șirului altfel returnează null. 

char a[20]="infobac";
char b[20] = strstr(a , "bac");
cout << b; // bac

Eliminarea și inserarea unui caracter într-un șir sunt operații frecvente și pot fi realizate cu ajutorul funcției strcpy. Deoarece comportamentul funcției strcpy este impredictibil dacă parametri se suprapun, este necesară utilizarea unui șir suplimentar. 

char s[256] = "informatica", t[256];
int pozitia = 2;

// eliminarea elementului de pe pozitia 2 din sirul s
strcpy(t , s + pozitia + 1); // copeim in sirul suplimentar caracterele incepand cu pozita 3
strcpy(s + pozitia , t); // copiem in sir incepand de la pozitia 2 caracterele din sirul suplimentar

// inserarea unui caracter in sir pe pozitia 2
strcpy(t , s + pozitia); // copeim in sirul suplimentar caracterele incepand cu pozita 2
strcpy(s + pozitia + 1 , t); // copiem in sir incepand de la pozitia 3 caracterele din sirul suplimentar
// in momentul de fata pe pozitia 2 nu exista un caracter
s[pozitia] = 'A'; // adaugam un caracter pe pozitia 2
