Pseudocod

Pseudocodul este un limbaj cu ajutorul căruia pot fi descriși algoritmii. Deși folosește convenții structurale asemănătoare cu cele ale unui limbaj de programare, pseudocodul a fost creat mai degrabă pentru a fi înțeles și interpretat de către oameni decât de către calculatoare.

Un program pseudocod nu poate fi interpretat sau compilat de către un calculator, dar poate fi cu ușurință “tradus” în limbaj de programare.
