Operatori

Operatorii sunt caractere speciale sau cuvinte cheie prin intermediul carora se reprezinta operatiile care se efectueaza in cadrul algoritmului.

1. Operatorul de atribuire: variabila ← expresie

x ← 2
y ← x + 5
x ← x + 1

Mai intai se evalueaza expresia din partea dreapta a sagetii, apoi valoarea acesteia este retinuta de variabila specificata în partea stanga.

2. Operatori matematici: + (adunare), – (scadere), * (inmultire), / (catul impartirii), % (restul impartirii sau mod).

Operatorii matematici se aplica pe date de tip numeric iar rezultatul va fi numeric. 

1 + 2 = 3
4 - 2 = 2
2 * 3 = 6
9 / 4 = 2
5 % 2 = 1

3. Operatori relationali: < (mai mic), > (mai mare), <= (mai mic sau egal), >= (mai mare sau egal), = (egal), ≠ (diferit). 

(5 < 10) este Adevarat
(1 = 1) este Adevarat
(2 >= 5) este Fals
(6 >= 6) este Adevarat
(23 ≠ 23) este Fals

Operatorii relationali se aplica pe date de tip numeric sau sir de caractere iar rezultatul este de tip logic (adevarat sau fals). A nu se confunda operatorul de atribuire ← cu operatorul relational = care doar verifica egalitatea intre doua variabile.

4. Operatori logici: si, sau. Operatorul "si" are rezultat adevarat cand toti operanzi sunt adevarati, operatorul "sau" are rezultat adevarat cand cel putin unul din operanzi este adevarat. 

(5 < 10) si (1 = 1) este Adevarat
(1 ≠ 1) si (5 > 10) este Fals
(1 = 1) si (1 > 2) este Fals
(1 = 1) sau (1 > 2) este Adevarat
(1 ≠ 1) sau (5 > 10) este Fals

Operatorii logici se aplica pe date de tip logic iar rezultatul este tot de tip logic.
