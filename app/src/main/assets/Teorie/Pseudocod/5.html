Conversia structurilor repetitive

Unul dintre exercițiile de bacalaureat cere conversia unei structuri repetitive intr-o altă structură repetitivă.

Spre exemplu modificare unei structuri repetitive cu test inițial (cât timp) într-o structură repetitivă cu test final (repetă până când). 

┏━ cat timp x ≠ 0 executa
┃     x ← x - 1 
┃     scrie x
┗━■

Devine:

┏━ daca x ≠ 0 atunci
┃┏━ repeta
┃┃     x ← x - 1 
┃┃     srcie x
┃┗━ pana cand x = 0
┗━■

Modificare unei structuri repetitive cu test final (repetă până când) într-o structură repetitivă cu test inițial (cât timp). 

┏━ repeta
┃     x ← x - 1 
┃     srcie x
┗━ pana cand x = 0

Devine:

x ← x - 1 
scrie x
┏━ cat timp x ≠ 0 executa
┃     x ← x - 1 
┃     scrie x
┗━■

Modificarea unei structuri repetitive cu număr cunoscut de pași (pentru) într-o structură repetitiva cu test inițial (cat timp). 

┏━ pentru i ← 0,n executa
┃ ┏━  daca i % 2 = 1
┃ ┃    scrie i
┃ ┗━■
┗━■

Devine:

i ← 0
┏━ cat timp i <= n executa
┃ ┏━  daca i % 2 = 1
┃ ┃    scrie i
┃ ┗━■
┃  i ← i + 1
┗━■
