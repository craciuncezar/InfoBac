Pentru a intelege structura unui program C++ vom analiza fiecare linie de cod a celebrului program "Hello, World!" ce face doar o simpla afisare pe ecran.

#include < iostream >
using namespace std;

int main()
{
&emsp; cout << "Hello World!";
&emsp; return 0;
}

Linia #include < iostream > include in program biblioteca iostream(InputOutputSTREAM) ce ne permite afisarea pe ecran si citirea de la tastatura.
Linia using namespace std; transmite compilatorului ca vom folosi namespaceul std(standard). Namespaceul std include toate functionalitatile biblioteci standard C++.

<b>Observatie!</b>
Linile goale si spatiile sunt ignorate, asadar acestea sunt folosite pentru a imbunatatii citirea codului.

Executia oricarui program incepe cu functia main, in cazul de fata int main().
Acoladele indica inceputul si sfarsitul unei functii. Instructiunile din interiorul acoladelor indica functionalitatea functiei (ce face aceasta functie).
Urmatoarea linie, cout << "Hello World!"; are ca scop afisarea pe ecran a mesajului "Hello, world!".

<b>Observatie!</b>
In C++ ; (punct si virgula) este folosit pentru a specifica sfarsitul unei instructiuni. Fiecare instructiune trebuie sa se termine cu ; . Aceasta indica sfarsitului unei expresii logice. Poti avea mai multe instructiuni pe aceasi linie, atata timp cat acestea se termina cu ; .

Ultima instructiune din program return 0; are ca scop returnarea valorii 0 si terminarea functiei main(), deci terminarea programului. Orice valoare returnata diferita de 0 (de obicei 1) semnalizeaza terminarea programului intr-un mod anormal.

<b>Observatie!</b>
Linia return 0; poate fi omisa declarand functia main ca void main().

<b>Exemplu:</b>

#include < iostream >
using namespace std;

void main()
{
&emsp;cout << "Hello World!";
&emsp;//lipseste return 0;
&emsp;/*comentariu
&emsp;pe mai multe linii*/
}

<b>Observatie!</b>
Comentariile reprezintă o succesiune de caractere pe care compilatorul nu le ia în considerare.
In C/C++ exista două tipuri de comentarii:
- pe mai multe linii cu marcajul /*…*/
- pe o singură linie (de la marcaj până la sf. de linie) cu marcajul //
Sunt utilizate pentru a creşte gradul de lizibilitate a programului şi ajută utilizatorii multiplii la înţelegerea programului.
