Structuri repetitive

Se pot defini mai multe tipuri de structuri repetitive în funcție de numărul de pași sau momentul verificării condiției.

1. Structura repetitivă cu test inițial: CAT TIMP (se folosește când numărul de repetiții este nedefinit) 

┏━ cat timp expresie executa
┃     instructiuni
┗━■

2. Structura repetitivă cu test final: REPETA-PANA CAND (se folosește când numărul de repetiții este nedefinit) 

┏━ repeta
┃     instructiuni
┗━ pana cand expresie

Structura repetitivă cu test final va fi exectuată cel puțin o dată.

3. Structura repetitivă cu număr cunoscut de pași: PENTRU (se folosește când numărul de repetiții este cunoscut – un număr fix de ori) 

┏━ PENTRU contor ← exp_init,exp_fin EXECUTA
┃     instructiuni
┗━■

Exemple de structuri repetitive:

Afișarea pe ecran a tuturor numerelor întregi pozitive mai mici decât un număr întreg primit ca input de la tastatură. 

citeste x (numar intreg)
┏━ cat timp x ≠ 0 executa
┃     x ← x - 1
┃     scrie x
┗━■

Afișarea pe ecran a tuturor numerelor întregi pozitive impare din intervalul [1,n], unde n este număr întreg citit de la tastatură. 

citeste n (numar intreg)
┏━ pentru i ← 0,n executa
┃ ┏━  daca i % 2 = 1
┃ ┃    scrie i
┃ ┗━■
┗━■
