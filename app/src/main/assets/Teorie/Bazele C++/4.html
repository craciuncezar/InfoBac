Instrucțiunea if

Programele C++ nu se limitează la o secvența liniară de instrucțiuni. În timpul procesului, un program poate repeta segmente de cod, sau poate lua decizii. C ++ oferă instrucțiuni de control al fluxului care servesc la specificarea a ceea ce trebuie făcut de programul nostru, când și în ce condiții.

Instrucțiunea "if" este utilizată pentru a executa o instrucțiune sau un bloc de instrucțiuni, dacă și numai dacă, o condiție este indeplinită. Sintaxa este următoarea: 

if(x > 0)
    cout<<"număr pozitiv";

Daca doriți să includeți mai mult de o singură instrucțiune pentru a fi executată atunci când condiția este îndeplinită, se folosește un bloc de instrucțiuni: 

if (x != 0)
{
   cout << "x este ";
   cout << x;
}

Instrucțiunea if poate, de asemenea, să specifice ce se intamplă atunci când condiția nu este îndemplinită, utilizând cuvântul cheie else. Sintaxa sa este: 

if (x == 0)
  cout << "x este 0";
else
  cout << "x este diferit de 0";

Tot odată poate lega mai multe condiții prin utilizarea mai multor if-uri. 

if (x > 0)
  cout << "x este pozitiv";
else if (x < 0)
  cout << "x este negativ";
else
  cout << "x este 0";
