Variabile și tipuri de date

Să ne imaginăm că trebuie să reți numărul 5 și numărul 2 în același timp. Ai reținut două valori diferite în memoria ta (5 și 2). Acum trebuie să adaugi 1 la primul număr. În momentul de față memorezi numerele 6 (adică 5 + 1) și 2 în memorie. Apoi calculezi suma celor două numere.

Întreg procesul de mai sus este similar cu ceea ce un calculator poate face cu două variabile. Așadar același proces poate fi exprimat in C++ prin următoarele instrucțiuni: 

a = 5;
b = 2;
a = a + 1;
suma = a + b;

O variabilă este un identificator, purtător a unei singure valori și se caracterizează prin: nume și tip de dată.

Fiecare variabilă are nevoie de un nume care o identifică și o deosebește de celelalte. De exemplu, în codul anterior numele variabilelor au fost a, b și sumă, dar am fi putut numi variabilele orice nume valabil în C++. Numele valabile în C++ nu pot conține spații, simboluri, semne de punctuație, să inceapă cu cifră, sau să existe cuvinte cheie in C++ cu aceași denumire. 

variabila mea = 5; // eroare
variabila.doi = 2; // eroare
int = 1; // eroare
1variabila = 3; // eroare
variabila1 = 3; // corect

Limbajul C/C++ face distincție între litere mari și mici. Variabilele NR, nR, nr, Nr sunt diferite din punctul de vedere al C/C++.

Tipuri de date elementare in C++:

int a = 5; // Număr întreg
float b = 2.5; // Număr real
unsigned c = 2; // Număr natural
char a = 'A'; // Caracter
double d = 1.0000008 // Număr real (pune accent pe partea fracționară)

Pentru fiecare variabilă se alocă (rezervă) un spațiu de memorie conform cu tipul ei. Acest lucru influențează domeniul de valori care se poate aloca variabilei. Pentru tipurile int și double se poate adăuga în față cuvântul long pentru a avea un domeniu de valori extins (long double sau long int).

Declararea mai multor variabile de același tip se poate face pe același rând: 

int a, b, c;

Inițializarea valorilor variabilelor la momentul declarării este opțională.
