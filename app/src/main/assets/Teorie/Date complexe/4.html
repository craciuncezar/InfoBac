Tipul struct

De multe ori datele folosite într-un program caracterizează obiecte/fenomene din realitate, iar acestea au mai multe caracteristici decât un simplu număr sau un șir de caractere. Pentru astfel de date folosim tipul struct.

Să ne imaginăm că vrem să gestionăm datele unui elev (nume, prenume, vârstă). În loc să folosim mai multe variabile pentru un elev putem să le grupăm pe toate sub un singur tip de date: 

struct Elev{
    char nume[21]; 
    char prenume[21];
    int varsta;
};

Declararea unei variabile din tipul definit poate fi făcută fie în momentul declarării structurii fie anterior. 

struct Elev{
    char nume[21]; 
    char prenume[21];
    int varsta;
} elev1, elev2; // declararea variabilelor aici

// sau ulterior

Elev elev3;

Accesarea unui câmp din structura se face cu ajutorul operatorului . (punct), numit și operator de acces: 

struct Elev{
    char nume[21]; 
    char prenume[21];
    int varsta;
};

Elev elev;
elev.nume = "Popescu";
elev.prenume = "Ionescu";
elev.varsta = 15;
cout<< "Elev: " << elev.nume << " " << elev.prenume << " cu varsta " << elev.varsta << "ani."; 
// afiseaza "Elev: Popescu Ionescu cu varsta 15 ani."
