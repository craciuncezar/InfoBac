Operatori

Odată introduse variabilele putem începe să modificăm variabilele și să stabilim relații folosind operatori.

1. Operatii aritmetice: + (adunare), - (scadere), * (inmultire), /(impartire), %(modulo)

Operațiile de adunare, scădere, înmulțire și împărțire corespund literalmente operatorilor lor matematici. Ultimul, operatorul modulo, reprezentat de un semn procentual (%), dă restul imparțiri a două valori. De exemplu: 

x = 11 % 3;
// x este 2 deoarece 11 impărțit la 3 
// are câtul 3 și restul 2

2.Operatori compuși: (+=, -=, *=, /=)

y += x; // echivalent cu y = y + x;
x -= 5; // echivalent cu x = x - 5;
x /= y;	// echivalent cu x = x / y;

3.Operatori de încrementare și decrementare: ++ și --

y++; // echivalent cu y += 1 și y = y + 1;
y--; // echivalent cu y -= 1 si y = y - 1;

Acești operatori de încrementare și decrementare pot fi folosiți pentru variabile și ca sufix și ca prefix, diferența fiind momentul evaluării expresiei: 

// Prefix
x = 3;
y = ++x;
// x este 4, y este 4

// Suffix
x = 3;
y = x++;
// x este 4, y este 3

4. Operatorii relaționali: == (egal), != (diferit), > (mai mare), < (mai mic), >= (mai mare sau egal), <= (mai mic sau egal) 

(7 == 5) // se evaluează fals
(5 > 4) // se evaluează adevarat
(3 != 2) // se evaluează adevarat
(6 >= 6) // se evaluează adevarat
(5 < 5) // se evaluează fals

Atenție! Operatorul de atribuire (operator =, cu un semn egal) nu este același cu operatorul de comparare a egalității (operator ==, cu două semne egal).

5.Operatori logici: ! (negație), && (și), || (sau)

Operatorul && "și" are rezultat adevărat când toți operanzi sunt adevarați, operatorul || "sau" are rezultat adevărat când cel puțin unul din operanzi este adevărat. 

(5 < 10) && (1 == 1) este Adevărat
(1 != 1) && (5 > 10) este Fals
(1 = 1) && (1 > 2) este Fals
(1 = 1) || (1 > 2) este Adevărat
(1 != 1) || (5 > 10) este Fals

Operatorul de negație inversează rezultatul unei expresii ex:

!(1 == 1) este Fals
!(1 != 1) este Adevărat

Ordinea operaților!!

O singură expresie poate avea mai mulți operatori. De exemplu:

x = 5 + 7 % 2;

În C ++, expresia de mai sus atribuie întotdeauna valoarea 6 variabilei x, deoarece operatorul % are o prioritate mai mare decât operatorul + și este întotdeauna evaluat înainte. Părți ale expresiilor pot fi puse între paranteze pentru a înlocui această ordine de prioritate sau pentru a clarifica în mod explicit efectul dorit. Observați diferența: 

x = 5 + (7 % 2); // x = 6 (ca fară paranteze)
x = (5 + 7) % 2; // x = 0

În ordinea scăderii importanței, operatorii sunt:
1) ! (negare)
2) *, /, %
3) +, –
4) <, <=, >=, >
5) ==, !=
6) &&
7) ||
