Citire și afișare

Exemplele de programe din secțiunile anterioare au oferit puțina interacțiune cu utilizatorul. Biblioteca standard oferă multe modalități suplimentare de a interacționa cu utilizatorul prin intermediul funcțiilor de intrare (input) / ieșire (output).

Instrucțiunea cin
Sintaxă: cin>>variabila1>>variabila2…
Efect: se preiau de la tastatură valori pentru fiecare variabilă din sir
Cerințe: folosirea instrucțiunii necesita #include <iostream>

Instrucțiunea cout
Sintaxă: cout << expresie1 << expresie2..;
Efect: se afișează, pe rând, valorile expresiilor
Cerințe: folosirea instrucțiunii necesită #include <iostream>
Pentru a trece pe urmatoarea linie se folosește expresia endl

Exemplu: program care citește de la tastatură două numere și afișează suma și produsul lor. 

#include <iostream>
using namespace std;

int main ()
{
    int a, b, suma, produs;
    cout<<"Introduceți cele două numere: ";
    cin>>a>>b;
    suma = a + b;
    produs = a * b;
    cout<<"Suma numerelor "<< a << " și " << b <<"este "<< suma << endl;
    cout<<"Produsul numerelor "<< a <<" și "<< b <<"este "<< produs << endl;
}
