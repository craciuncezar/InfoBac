Hello world!

Cea mai bună metodă de a învața un limbaj de programare este evident scrierea unui program. De obicei, in programare, începătorii pornesc prin scrierea unui program numit "Hello world!", care efectuează simpla afișare pe ecran a mesajului "Hello world!". 

// Primul program in C++
#include <iostream>
using namespace std;

int main ()
{
  cout << "Hello World! ";
}

Deși este un program foarte simplu, conține toate componentele fundamentale ale unui program C++. Haideți să analizăm programul linie cu linie.

Linia 1: // Primul meu program in C++
Cele două slash-uri "//" indică faptul că restul liniei este un comentariu introdus de către programator, dar care nu are niciun efect asupra comportamentului programului. Programatorii folosesc comentariile pentru a include scurte explicații sau observații referitoare la cod.

Linia 2: #include <iostream>
Liniile care încep cu un semn hash (#) sunt directive citite și interpretate de ceea ce este cunoscut sub numele de preprocesor. Sunt linii speciale interpretate înainte de începerea compilării programului. În acest caz, prin directiva #include <iostream> preprocesorul include în program instrucțiunile necesare pentru a citi și afișa pe ecran.

Linia 3: using namespace std;
Această linie permite utilizarea elementelor din biblioteca C++ standard fără a fi nevoie să specificăm locația acestora.

Linia 4 este o linie goala, ce nu are efect asupra programului și pur și simplu îmbunătățesc lizibilitatea codului.

Linia 5: int main ()
Această linie declară o funcție numită main. În esență, o funcție este un grup de instrucțiuni de cod. Funcțiile vor fi discutate în detaliu într-un capitol ulterior.

Funcția numită main este o funcție specială în toate programele C++, ea reprezintă începutul instrucțiunilor executate de către calculator, practic este punctul de start al programului.

Liniile 6 si 8: { si }
Acoladele indică un bloc de cod. Un bloc de cod reprezintă mai multe instrucțiuni grupate împreună, în cazul de față instrucțiunile din bloc reprezintă instrucțiunile executate de către funcția principală main.

Linia 7: cout << "Hello World! ";
Aceasta este o instrucțiune C++ ce afișează pe ecran mesajul "Hello World!".

Instrucțiunile sunt executate în ordinea în care sunt scrise. Observați că o instrucțiune se termină cu punct și virgulă ";". Acest caracter marchează sfârșitul instrucțiuni, la fel cum punctul "." marchează finalul unei propoziții în limba română. Una dintre cele mai comune erori de sintaxă în C++ este omiterea încheierii unei instrucțiuni prin punct și virgulă ";".
