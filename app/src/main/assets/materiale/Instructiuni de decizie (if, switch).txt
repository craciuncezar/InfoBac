<b>If [else]</b>

Sintaxa:
if (expresie)
&emsp;instr1;
[else (opţională) 
&emsp;instr2;]

Instrucţiunea evaluează expresia logică, dacă este adevărată (1) execută instr1 iar dacă nu execută instr2 dacă ramura de altfel este prezentă iar în caz contrar nu execută nimic.

<b>Exemplu:</b> 
Determinarea valorii maxime dintre 2 valori întregi a şi b citite (cu ramură de else)

int a,b;
cin >> a >> b;
if (a > b)
&emsp;cout << a;
else
&emsp;cout << b;

<b>Switch</b>
Se utilizează atunci când decizia se ia în urma evaluării mai multor expresii (şi este unică)

Sintaxa:
switch (expresie) 
{	
&emsp;case&emsp;constanta_1: instr_1; break;
&emsp;…
&emsp;case&emsp;constanta_n : instr_n; break;
&emsp;[default: instr;]
}

<b>Observaţii!</b>

-expresie nu poate avea alt tip decât întreg sau character.
-constantele pot să se găsească în valoarea expresiei sau nu 
-caz în care se execută instrucţiunea de pe ramura default (dacă există , altfel nu se execută nimic)
-dacă instrucţiunea break (ieşire necondiţionată) nu există atunci se vor executa toate instrucţiunile de la prima constantă egală cu valoarea expresiei.
