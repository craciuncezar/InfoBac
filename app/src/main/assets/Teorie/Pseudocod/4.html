Structuri repetitive

Se pot defini mai multe tipuri de structuri repetitive in functie de numarul de pasi sau momentul verificarii conditiei.

1. Structura repetitiva cu test initial: CAT TIMP (se foloseste cand numarul de repetitii este nedefinit) 

┏━ cat timp expresie executa
┃     instructiuni
┗━■

2. Structura repetitiva cu test final: REPETA-PANA CAND (se foloseste cand numarul de repetitii este nedefinit) 

┏━ repeta
┃     instructiuni
┗━ pana cand expresie

Structura repetitiva cu test final va fi exectuata cel putin o data.

3. Structura repetitiva cu numar cunoscut de pasi: PENTRU (se foloseste cand numarul de repetitii este cunoscut – un numar fix de ori) 

┏━ PENTRU contor ← exp_init,exp_fin EXECUTA
┃     instructiuni
┗━■

Exemple de structuri repetitive:

Afisarea pe ecran a tuturor numerelor intregi pozitive mai mici decat un numar intreg primit ca input de la tastatura. 

citeste x (numar intreg)
┏━ cat timp x ≠ 0 executa
┃     x ← x - 1
┃     scrie x
┗━■

Afisarea pe ecran a tuturor numerelor intregi pozitive impare din intervalul [1,n], unde n este numar intreg citit de la tastatura. 

citeste n (numar intreg)
┏━ pentru i ← 0,n executa
┃ ┏━  daca i % 2 = 1
┃ ┃    scrie i
┃ ┗━■
┗━■
