Operatori

Operatorii sunt caractere speciale sau cuvinte cheie prin intermediul cărora se reprezintă operațiile care se efectuează în cadrul algoritmului.

1. Operatorul de atribuire: variabila ← expresie

x ← 2
y ← x + 5
x ← x + 1

Mai întâi se evaluează expresia din partea dreaptă a sageții, apoi valoarea acesteia este reținută de variabila specificată în partea stanga.

2. Operatori matematici: + (adunare), – (scadere), * (inmultire), / (catul impartirii), % (restul impartirii sau mod).

Operatorii matematici se aplică pe date de tip numeric iar rezultatul va fi numeric. 

1 + 2 = 3
4 - 2 = 2
2 * 3 = 6
9 / 4 = 2
5 % 2 = 1

3. Operatori relaționali: < (mai mic), > (mai mare), <= (mai mic sau egal), >= (mai mare sau egal), = (egal), ≠ (diferit). 

(5 < 10) este Adevărat
(1 = 1) este Adevărat
(2 >= 5) este Fals
(6 >= 6) este Adevărat
(23 ≠ 23) este Fals

Operatorii relaționali se aplică pe date de tip numeric sau șir de caractere iar rezultatul este de tip logic (adevărat sau fals). A nu se confunda operatorul de atribuire ← cu operatorul relațional = care doar verifică egalitatea între două variabile.

4. Operatori logici: și, sau. Operatorul "și" are rezultat adevărat când toti operanzi sunt adevărați, operatorul "sau" are rezultat adevărat când cel puțin unul din operanzi este adevărat. 

(5 < 10) și (1 = 1) este Adevărat
(1 ≠ 1) si (5 > 10) este Fals
(1 = 1) si (1 > 2) este Fals
(1 = 1) sau (1 > 2) este Adevărat
(1 ≠ 1) sau (5 > 10) este Fals

Operatorii logici se aplică pe date de tip logic iar rezultatul este tot de tip logic.
