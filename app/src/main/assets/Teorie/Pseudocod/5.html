Conversia structurilor repetitive

Unul dintre exercitiile subiectului 1 de bacalaureat cere conversia unei structuri repetitive intr-o alta structura repetitiva.

Spre exemplu modificare unei structuri repetitive cu test initial (cat timp) intr-o structura repetitiva cu test final (repeta pana cand). 

┏━ cat timp x ≠ 0 executa
┃     x ← x - 1 
┃     scrie x
┗━■

Devine:

┏━ daca x ≠ 0 atunci
┃┏━ repeta
┃┃     x ← x - 1 
┃┃     srcie x
┃┗━ pana cand x = 0
┗━■

Modificare unei structuri repetitive cu test final (repeta pana cand) intr-o structura repetitiva cu test initial (cat timp). 

┏━ repeta
┃     x ← x - 1 
┃     srcie x
┗━ pana cand x = 0

Devine:

x ← x - 1 
scrie x
┏━ cat timp x ≠ 0 executa
┃     x ← x - 1 
┃     scrie x
┗━■

Modificare unei structuri repetitive cu numar cunoscut de pasi (pentru) intr-o structura repetitiva cu test initial (cat timp). 

┏━ pentru i ← 0,n executa
┃ ┏━  daca i % 2 = 1
┃ ┃    scrie i
┃ ┗━■
┗━■

Devine:

i ← 0
┏━ cat timp i <= n executa
┃ ┏━  daca i % 2 = 1
┃ ┃    scrie i
┃ ┗━■
┃  i ← i + 1
┗━■
