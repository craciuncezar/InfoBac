Ce sunt limbajele de programare?

Un limbaj de programare facilitează comunicarea dintre oameni și calculatoare. Așadar ne permite să dăm instrucțiuni calculatorului, într-un limbaj pe care acesta îl ințelege. Așa cum oamenii pot vorbi mai multe limbi și calculatorul poate executa instrucțiuni în mai multe limbaje de programre.

Calculatoarele ințeleg un limbaj numit binar, compus din cifre binare ce pot avea valoarea 1 sau 0. Cu toate acestea programatorii nu scriu limbaj binar ci folosesc limbaje de programare precum C++, Java sau Python care mai apoi sunt traduse în acest limbaj binar, acest proces fiind numit "compilare".

La fel ca în cazul limbajelor vorbite de oameni, limbajele de programare au propriul set de reguli de vocabular și de gramatică. Vocabularul limbajului de programare este definit de un set de cuvinte cheie, iar gramatica este denumită sintaxă.
